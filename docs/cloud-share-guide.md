# Lightsee Cloud Share 사용 가이드

마크다운 파일이나 폴더를 인터넷에 공유하는 방법을 안내합니다.

---

## 1. 데스크톱 앱에서 파일 공유하기

1. Lightsee 앱을 실행합니다.
2. 공유하고 싶은 마크다운 파일을 엽니다.
3. 상단 툴바에서 **Cloud** 버튼을 클릭합니다.
4. **Cloud Share** 버튼을 클릭하면 업로드가 시작됩니다.
5. 업로드가 완료되면 공유 URL이 표시됩니다.
6. **Copy URL** 버튼을 눌러 링크를 복사합니다.
7. 복사한 링크를 메신저, 이메일 등으로 전달하면 됩니다.

> 받는 사람은 별도 설치 없이 웹 브라우저에서 바로 열람할 수 있습니다.

### 공유 업데이트 / 삭제

- **Update** — 파일을 수정한 후 눌러주면 공유 링크는 그대로 유지되고 내용만 갱신됩니다.
- **Remove** — 공유를 중단하고 싶을 때 누르면 링크가 비활성화됩니다.

---

## 2. 터미널에서 폴더 공유하기

여러 파일이 들어 있는 폴더를 한번에 공유할 수 있습니다.

### 사전 준비 (최초 1회)

터미널을 열고 아래 명령어를 실행합니다:

```
sudo ln -s /Applications/Lightsee.app/Contents/MacOS/lightsee /usr/local/bin/lightsee
```

비밀번호를 입력하면 설정이 완료됩니다.

### 폴더 공유

```
lightsee expose ./내폴더
```

업로드가 완료되면 공유 URL이 출력됩니다:

```
Uploaded 12 file(s). Share URL: https://lightsee.vercel.app/s/abc123
```

이 URL을 전달하면 됩니다. 받는 사람은 폴더 구조를 탐색하며 각 문서를 볼 수 있습니다.

### 단일 파일 공유

```
lightsee expose ./README.md
```

### 공유 목록 확인

```
lightsee list
```

### 공유 삭제

```
lightsee remove abc123
```

`abc123` 부분은 `lightsee list`에서 확인할 수 있는 공유 ID입니다.

### 내용 업데이트

파일을 수정한 후 같은 명령어를 다시 실행하면 됩니다:

```
lightsee expose ./내폴더
```

링크는 바뀌지 않고 내용만 갱신됩니다.

## 자주 묻는 질문

**Q. 공유 링크에 비밀번호를 설정할 수 있나요?**
A. 현재는 링크를 아는 사람이면 누구나 볼 수 있습니다.

**Q. 공유한 파일은 어디에 저장되나요?**
A. 클라우드 서버에 저장됩니다. 삭제하기 전까지 유지됩니다.

**Q. 파일을 수정하면 자동으로 반영되나요?**
A. 아닙니다. 수정 후 다시 공유(Update 또는 `lightsee expose`)해야 반영됩니다.

**Q. 마크다운 파일 외에 다른 파일도 공유할 수 있나요?**
A. 마크다운 파일(.md)만 지원합니다.

**Q. 한번에 몇 개까지 공유할 수 있나요?**
A. 폴더당 최대 200개 파일, 파일당 최대 5MB입니다.
